# 数据质量评估报告 — 空间分析研究者视角

> 视角：空间分析 / 生物地理研究者
> 评估对象：全量降采样数据集 `occurrence_clean`（3,997,847 条）及其网格聚合
> 评估时间：2026-06-11
> 方法：直接对生产库运行空间统计查询（大洲密度、网格覆盖、聚集度、纬度分布、坐标质量、物种丰富度、时间跨度）

---

## 一句话结论

**这是一份坐标质量高、物种覆盖近乎完整、空间偏倚已被大洲级配额有效纠正的"展示级"数据集**；适合做物种分布范围、相对热点、丰富度制图与全球热力可视化，**但不适合做绝对种群丰度估计、跨大洲丰度对比、全年物候/迁徙分析**——后三者受"单年 4 个月 + 降采样语义"根本性限制。

## 亮点

1. **空间偏倚纠正达到设计目标**。降采样后欧洲、北美、南美、大洋洲密度几乎相同（约 366 条/万km²），证明"0.1° 去重 + 大洲面积配额"成功削平了 eBird 在欧美的观察者过采样。亚洲、非洲因 Step1 产出本就低于配额而全量保留，保留了它们真实偏低的采样密度——**这是正确取舍**，避免了对稀疏区的虚构填充。
2. **坐标质量高**。`COUNTRY_COORDINATE_MISMATCH`(0.03%)、`PRESUMED_SWAPPED_COORDINATE`(0.0005%)、`ZERO_COORDINATE`(0) 三类硬错误合计 <0.04%，无 null island。可直接用于空间叠加分析而几乎无需再清洗。
3. **物种覆盖近乎完整**。9,807 个物种 key（约全球鸟类总数的 89%），且 99% 记录有种级鉴定。单格物种丰富度均值 89、最高 708，符合从温带到热带的纬度多样性梯度。
4. **网格聚合口径与 API 实时聚合一致**（见 [后端/WebGIS 评估报告](2026-06-11_backend_webgis_data_strategy.md)），制图与查询不会错位。

## 问题（按对分析的影响排序）

### P1（高）：时间维度极窄 —— 单年 2024、仅 8–11 月
- 仅覆盖**北半球秋季迁徙窗**。缺春季迁徙（3–5 月）、繁殖季、越冬季。
- 直接后果：`/stats/migration`（月度观测重心）只能呈现秋季南向位移的片段，**无法表达完整迁徙周期**；任何物候、季节性、年际趋势分析均不成立。
- 南半球此窗为春季，南北半球的"季节语义"相反，跨半球的月度对比需谨慎解释。

### P2（高）：降采样语义 ≠ 丰度，`individual_sum` 不可读作种群数量
- Step1 对每 (0.1°格 × 物种 × 月) 仅保留 1 条（max gbifID）。因此 `individual_count` 是该格该种**某一份清单**的计数，网格 `individual_sum` 是这些代表值之和，**不是种群总量**。
- `record_count` 是"物种×0.1°子格"的出现数（经大洲配额封顶），是**occupancy/richness 代理**，不是观察努力或丰度。
- 制图与前端 tooltip 必须避免"该区域有 N 只鸟"的表述，应表述为"观测记录数 / 物种出现强度"。

### P3（中）：1° 等经纬网格的面积失真（MAUP / 极向收缩）
- 1° 格子面积随 cos(纬度) 收缩：50°N 处约为赤道格的 0.64×。`record_count` 未做面积归一，高纬格在密度配色上会**虚高**。
- 全球热力图建议按格面积归一（条/km²）或改用等积格网（如 ISEA/H3）做密度对比；当前网格更适合"计数"而非"密度"语义。

### P4（中）：亚大洲级聚集未被打散
- 前 1% 格子仍占 16% 记录，单格最高 1.8 万。城市与观鸟热点（如某些海岸湿地）在区域尺度仍主导。区域级密度对比、点格局（point pattern）分析需意识到这是观察者驱动的聚集，非生态信号。

### P5（低）：有效空间分辨率上限为 0.1°
- Step1 去重把最细呈现粒度锁定在 0.1°（约 11km @ 赤道）。**任何 <0.1° 的点格局、最近邻、核密度分析无效**；`/occurrence/points` 在高缩放级别展示的是"每格每种每月一点"的稀疏代表点，非原始观测密度。

### P6（低）：跨大洲丰度对比无效
- 密度已被人为封顶到 ~366，且 Africa/Asia 低于此值反映的是采样不足而非鸟少。**跨大洲不能比"谁的鸟多"**，只能比分布范围与物种组成。

## 改进建议（按优先级）

| 优先级 | 建议 | 说明 |
|-------|------|------|
| 🔴 | 前端/文档明确标注数据的**时间窗（2024 年 8–11 月）与"记录数≠丰度"** | 防止误读，是科学诚信底线 |

## 适用性预估（fitness-for-use）

| 分析任务 | 可行性 | 备注 |
|---------|--------|------|
| 物种分布范围 / 出现热点制图 | ✅ 适用 | 0.1°–1° 尺度可靠 |
| 物种丰富度（α多样性）制图 | ✅ 适用 | 每格物种数指标稳健 |
| 全球热力可视化（相对强度）| ✅ 适用 | 建议面积归一 |
| 区域内热点识别 | 🟡 谨慎 | 含观察者偏倚，非纯生态 |
| 秋季南向迁徙趋势（8–11 月）| 🟡 有限可行 | 仅此窗，重心法非轨迹 |
| 绝对种群丰度估计 | ❌ 不适用 | 降采样语义 + null 缺失 |
| 跨大洲丰度对比 | ❌ 不适用 | 密度被封顶，见 P6 |
| 全年物候 / 年际趋势 | ❌ 不适用 | 单年 4 个月 |
| <0.1° 精细点格局 | ❌ 不适用 | 受 Step1 去重限制 |

**样本量预估**：4M 记录 / 8,720 格 / 9,807 种 → 平均 408 条/种，但高度右偏（首位物种 3.2 万条，长尾大量物种 <10 条）。可支撑常见种的分布制图与 1° 热力；稀有种的物种分布模型（SDM）样本不足，需谨慎或排除。
